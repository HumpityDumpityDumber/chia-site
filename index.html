Understanding .chia Syntax

Block type

Specifies the kind of configuration you are defining, such as a package, service, profile, or network.

package "discord"
services "waybar"

Comments

Comments are writen between two --

Organized Configuration

Chia uses a structured approach to configuration management:

  • chia.chia: Main configuration file that defines all profiles and network configuration
  • Packages: Separate user and system package definitions
  • Services: Declare all your systemd services
  • Configs: The configs folder holds your config rendering files for each app
